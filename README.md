# C++ Examples
Just for learning, testing and fun...

# Let's start...

## Requirements
* compiler with c++14 support
* [cmake](https://cmake.org)

## Configure

    git clone [https://github.com/w07c1ech/examples.git](https://github.com/w07c1ech/examples.git)
    cd examples
    mkdir build && cd build
    cmake . ..

## Build and run

    cd build
    make
    ./examples # or ctest
